# VSMW-frontend
Frontend side of VSMW application

## Grabbing the code
```
git clone https://github.com/...
```

## Environment
```
node.js version >= 8
npm version >= 5
```

## Setup dependencies 
```
npm install
```

## Running application
```
npm run start
```

visit the page after the start: [http://127.0.0.1:3000](http://127.0.0.1:3000)
